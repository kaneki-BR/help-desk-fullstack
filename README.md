# 🎫 Sistema de Chamados (Help Desk)

Um projeto full-stack desenvolvido para simplificar e organizar o registro, acompanhamento e resolução de tickets de suporte de TI. 

Este sistema foi construído não apenas como uma ferramenta de gestão, mas como um projeto marco para simbolizar a transição da vivência diária no suporte técnico (CPD) para a criação de soluções em desenvolvimento de software. A arquitetura foi desenhada separando as responsabilidades entre uma Web API robusta e uma interface de usuário dinâmica.

## 🚀 Tecnologias Utilizadas

**Back-end:**
* **C# / .NET 8:** Construção da Web API.
* **Entity Framework Core:** ORM utilizado para o mapeamento objeto-relacional (Code-First).
* **SQL Server (SSMS):** Banco de dados relacional.
* **Swagger:** Documentação e teste das rotas da API.

**Front-end:**
* **React:** Biblioteca JavaScript para construção da interface.
* **Vite:** Ferramenta de build super rápida para o React.
* **Bootstrap**

## ⚙️ Funcionalidades Principais (Em desenvolvimento)

- [x] Modelagem do banco de dados com EF Core.
- [ ] Criação, leitura, atualização e exclusão (CRUD) de chamados via API.
- [ ] Interface para abertura de novos tickets de suporte.
- [ ] Painel (Dashboard) para visualização de chamados abertos, em andamento e concluídos.
